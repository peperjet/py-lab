# py-lab

## 프로젝트 소개
본 폴더는 인프런 파이썬 강의를 수강하며  
기초 문법부터 실습 예제까지 학습한 내용을 정리한 실습 기록 공간입니다.

강의 내용을 단순히 따라 치는 것이 아니라,  
직접 실행하고 수정하며 이해한 코드를 중심으로 정리하였습니다.

---

## 학습 내용

- Python 기본 문법
  - 변수, 자료형
  - 조건문, 반복문
  - 함수

- 자료구조
  - 리스트, 튜플
  - 딕셔너리
  - 집합(Set)

- 파일 입출력
- 예외 처리
- 모듈과 패키지 사용
- 간단한 실습 프로젝트

---

## 학습 목적

- 파이썬 기초 문법의 체계적인 이해
- 실습 중심 반복 학습을 통한 코드 작성 능력 향상
- 이후 데이터 분석 및 백엔드 학습을 위한 기초 체력 확보

---

## 학습 방식

- 강의 수강 후 예제 코드 직접 재작성
- 코드 수정 및 확장 실습
- 오류 발생 시 디버깅 과정 기록
